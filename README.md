# CZL EXPRESS-TrackBox0.1
轨迹查询嵌入框，CZL Express苍正国际快递所用，可以在任意页面嵌入一个轨迹查询框，输入单号查询后通过get拼接查询链接，跳转到轨迹页面(**指定前部链接+?query=单号**）展示结果
